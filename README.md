# Animation Play Space

This is a simple Unreal Engine 5 project dedicated to creating various animation adjacent gameplay systems.

## Current Systems:

* Character locomotion animation.
  * Includes starts and stops
  * Generic functionality to allow realtime setting of animation sets.
  * Distance matching on stop animations.
  * Stride warping.
  * Configurable animation sets for easy swapping of animations.
  * Integrated upperbody animation set blending.
  * Configurable equipped item animation sets.
 
* GAS player abilities
  * Basic player attribute set.
  * Generic player dodge ability.
  * Stamina regen and regen delay gameplay effects.
  * Item equip ability.
 
* Character UI MVVM
  * MVVM wrapper to allow code side creation and setting of UI VMs.
  * Player vitals UI updated by VM.
